# Airline SQL Analysis Project

This project demonstrates SQL data analysis skills using an airline database.  
The passengers and flights are sample data used for analytical purposes.

---

## Database Structure

**Tables:**

- `Passenger` – contains passenger information  
- `Trip` – contains flight routes  
- `Pass_in_trip` – links passengers to trips with dates  

---

## Skills Demonstrated

- SQL JOINs  
- GROUP BY and aggregation functions  
- Filtering with WHERE and HAVING  
- Ordering results with ORDER BY  
- Complex queries with GROUP_CONCAT (finding co-passengers)  
- Analytical thinking and data exploration  

---

## Example Analysis Performed

1. Number of flights per passenger  
2. Most popular destinations (by passengers)  
3. Passengers who flew more than once  
4. Number of passengers per flight  
5. Departure cities with the most flights  
6. Full flight history of each passenger  
7. Flights of a selected passenger  
8. Passengers who flew to a specific city  
9. Number of flights per city per passenger  
10. Passengers flying together on the same flight  

---

## Project Files

- `database_schema.sql` – database tables and sample data  
- `queries.sql` – SQL queries and analysis  
- `README.md` – project description  

---

## How to Use

1. Create the database using `database_schema.sql`  
2. Run the queries in `queries.sql`  
3. Explore results and modify queries for further analysis  

---

## Technologies

- SQL (MySQL/PostgreSQL/SQLite)  
- GitHub for version control and portfolio showcase  

---

**This project is ideal for showcasing SQL and data analysis skills to recruiters and hiring managers.**



# Aviakompaniya SQL Təhlili Layihəsi

Bu layihə aviakompaniya verilənlər bazası üzərində SQL bacarıqlarını nümayiş etdirir.  
Sərnişinlər və uçuşlar analitik məqsədlər üçün nümunə məlumatlardır.

---

## Verilənlər Bazası Struktur

**Cədvəllər:**

- `Passenger` – sərnişin məlumatları  
- `Trip` – uçuş marşrutları  
- `Pass_in_trip` – sərnişinləri uçuşlarla əlaqələndirir (tarix daxil olmaqla)  

---

## Bacarıqlar

- SQL JOIN-lər  
- GROUP BY və toplama funksiyaları  
- WHERE və HAVING ilə filter  
- ORDER BY ilə nəticələrin sıralanması  
- GROUP_CONCAT ilə mürəkkəb sorğular (birgə sərnişinləri tapmaq)  
- Analitik düşüncə və məlumat araşdırması
